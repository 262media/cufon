Fonts

Note: for now only TrueType (TTF) and OpenType (OTF) fonts are supported.

Glyphs
Includes all available glyphs. Highly unrecommended.
Basic Latin uppercase letters (A-Z). (26 glyphs)
Basic Latin lowercase letters (a-z). (26 glyphs)
Basic Latin digits (0-9). (10 glyphs)
Basic Latin punctuation (!@#%...). (33 glyphs)
Basic Latin glyphs within the Unicode range 0x0021 to 0x007E. (94 glyphs)
The Missing Glyph (usually a box or a question mark) will only be displayed if you use a character that isn't supported by the font (or its own glyph wasn't included). By default these characters will not be displayed at all. Note: Not all fonts have this glyph.

The more glyphs you select the larger the resulting file will grow.

Security
(ex. example.org)

Separate multiple domains by either commas or spaces. Please make sure to include your domain both with and without «www» if necessary (if you're not sure, it's probably necessary).

Note: This is not a fool-proof solution but will prevent people from hotlinking to your fonts.

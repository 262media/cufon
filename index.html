cufón - fonts for the people

First, get the latest version of cufón.
(an uncompressed version is also available)

Font

Note: for now only TrueType (TTF) and OpenType (OTF) fonts are supported.

Glyphs
Includes all available glyphs. Highly unrecommended.
Basic Latin uppercase letters (A-Z). (26 glyphs)
Basic Latin lowercase letters (a-z). (26 glyphs)
Basic Latin digits (0-9). (10 glyphs)
Basic Latin punctuation (!@#%...). (33 glyphs)
Basic Latin glyphs within the Unicode range 0x0020 to 0x007E. (95 glyphs)
Latin glyphs within the Unicode range 0x00A0 to 0x00FF. Includes punctuation, currency symbols, numerals and accented characters (96 glyphs)
Extended Latin glyphs within the Unicode range 0x0100 to 0x017F. (128 glyphs)
Extended Latin glyphs within the Unicode range 0x0180 to 0x024F. (208 glyphs)
Combined glyphs, such as ffi and ch. Unsupported. More information
The Missing Glyph (usually a box or a question mark) will only be displayed if you use a character that isn't supported by the font (or its own glyph wasn't included). By default these characters will not be displayed at all. Note: Not all fonts have this glyph. Unsupported.

The more glyphs you select the larger the resulting file will grow.

Security
(ex. example.org)

Separate multiple domains by either commas or spaces. Please make sure to include your domain both with and without «www» if necessary (if you're not sure, it's probably necessary).

Note: This is not a fool-proof solution but will prevent people from hotlinking to your fonts.

Customization

Unless you plan on heavily customizing cufón, you should leave this value as it is.

Performance

A lower value (64 at minimum) results in better performance and smaller file size at the cost of accuracy. Values lower than 100 (sometimes more) are not recommended unless you are OK with visible loss of detail. The maximum value is 2048.
